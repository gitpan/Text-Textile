h1. Text::Textile - a Perl implementation of Textile

h2. DESCRIPTION

Text::Textile is a Perl impementation of the Textile formatting language.


h2. DOCUMENTATION

Documentation for this module is available in Textile.pm in POD format.


h2. INSTALLATION

Traditional Perl module installation applies here:

bc. perl Makefile.PL
make
make test
make install
